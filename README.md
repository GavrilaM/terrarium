# 🌿 Terrarium — Digital Ecosystem

> **AI-driven digital ecosystem where creatures live, evolve, and adapt in real-time.**

Built for the [Mayar Vibecoding Competition Ramadan 2026](https://mayar.id/vibe2026).

## 🧬 What is Terrarium?

Terrarium is a browser-based digital ecosystem simulation. Watch as AI-driven creatures are born, hunt for food, reproduce, mutate, and evolve — all happening in real-time in your browser.

### Features
- 🧫 **Procedural Creatures** — Every creature is uniquely generated from its DNA
- 🔄 **Real-time Evolution** — Genetic crossover & mutation across generations
- 🌊 **Living Ecosystem** — Food chains, predator-prey dynamics, natural selection
- ✨ **Bioluminescent Aesthetics** — Mesmerizing visual experience
- 🎮 **Interactive** — Add food, change climate, introduce new species
- 💰 **Mayar Integration** — Premium biomes & species via Mayar payment

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/GavrilaM/terrarium.git

# Serve locally
npx -y serve .

# Open http://localhost:3000
```

No build step needed — pure vanilla HTML/CSS/JS.

## 🛠️ Tech Stack

- **HTML5 Canvas** — 2D rendering engine
- **Vanilla JavaScript** — Zero frameworks, maximum performance
- **CSS3** — Glassmorphism UI, dark theme
- **Mayar API** — Payment integration
